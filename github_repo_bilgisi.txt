PROJE ADI: Hayvan Sınıflandırıcı Yapay Zeka Modeli

GITHUB REPOSU:
https://github.com/sebnemkrmnn/animal-classifier

PROJE AÇIKLAMASI:
Bu proje, görüntü sınıflandırmaya yönelik bir derin öğrenme modelini içerir.
Kullanıcıdan alınan bir hayvan görseli, eğitilmiş yapay zeka modeli ile analiz
edilerek doğru hayvan sınıfına atanır.

ANA DOSYALAR:
- app.py                 : Streamlit tabanlı kullanıcı arayüzü
- train_model.ipynb      : Modelin eğitildiği Jupyter Notebook dosyası
- simple_cnn_animals.pth : Eğitilmiş modelin kaydedilmiş hali
- data_preprocess.ipynb  : Veri hazırlama ve görselleştirme
- translate.py           : Sınıf etiketlerini yazıya çevirir
- requirements.txt       : Kullanılan Python kütüphaneleri listesi
- README.md              : Proje bilgilerini içeren açıklama dosyası

KULLANILAN TEKNOLOJİLER:
- Python
- PyTorch
- Torchvision
- Streamlit
- Pillow
- Jupyter Notebook
- TQDM

VERİ SETİ:
- Animals-10 veri seti (Kaggle üzerinden edinilmiştir)
- Görseller 'raw-img' klasöründe kategorilere göre klasörlenmiştir.
